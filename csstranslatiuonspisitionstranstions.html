Entendendo CSS Transitions

As CSS Transitions permitem que você adicione animações suaves quando as propriedades CSS de um elemento mudam. Isso melhora a experiência do usuário, tornando mudanças visuais mais agradáveis.

Como usar CSS Transitions?

Adicione a propriedade transition ao elemento que deseja animar. Ela define:

  • Propriedade: Qual propriedade CSS será animada (ex.: background-color, transform).
  • Duração: O tempo que a transição levará (ex.: 0.5s).
  • Função de Timing: O ritmo da animação (ex.: ease, linear).
  • Atraso: Quanto tempo esperar antes de começar (opcional).

Exemplo Prático

Este quadrado mudará de tamanho e cor quando o mouse passar por cima:

Código CSS:

.box {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
    transition: all 0.5s ease;
}
.box:hover {
    transform: scale(1.2);
    background-color: #8BC34A;
}

Principais Propriedades de Transition

  • transition-property: Define quais propriedades serão animadas (ex.: background-color, all).
  • transition-duration: Define a duração da animação (ex.: 1s, 500ms).
  • transition-timing-function: Controla o ritmo da animação (ex.: ease, linear, ease-in).
  • transition-delay: Define o atraso antes da animação começar (ex.: 0.2s).

Resumo

Com CSS Transitions, você pode criar efeitos visuais suaves ao alterar propriedades CSS. Teste diferentes combinações de transition para explorar suas possibilidades!
